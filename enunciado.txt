Desafio de codificação do desenvolvedor de pilha completa
Instruções Gerais
Objetivo: Criar uma API REST que registre e liste Organizações, Usuários e

Adesão de usuários a organizações.



Requisitos



As informações a seguir são necessárias para o registro e pesquisas de

cada entidade.



Organização:

● Nome



Do utilizador:

● Nome

● E-mail



Filiação:

● Usuário

● Organização

● Função



As seguintes operações devem estar disponíveis por meio da API:

X Deve ser capaz de registrar uma organização

X Deve ser capaz de retornar uma lista com todas as organizações

X Deve ser capaz de retornar uma única organização com um ID especificado

X Deve ser capaz de registrar um usuário

X Deve ser capaz de retornar uma lista de todos os usuários

X Deve ser capaz de retornar um único usuário com um ID especificado

X Deve ser capaz de registrar um usuário como membro de um determinado

Organização

● Deve ser capaz de retornar uma lista de membros de um determinado

Organização



Para esta tarefa, gostaríamos que você usasse as seguintes ferramentas:

● Framework sem servidor preferencialmente com Typescript

● Banco de dados DynamoDB



Bônus

Você pode mostrar sua habilidade com essas conquistas extras se quiser:

● Use um design de tabela única no DynamoDB

● Use paginação para resultados de lista

● Gravar testes para a API

● Erros de validação de entrada detalhados

● Documentação da API gerada automaticamente

● Autenticação usando OAuth2

● Crie funções de usuário separadas com aplicação de autorização adequada em

a API:

○ Administrador do sistema: autorizado a criar/listar Organizações, Usuários

e adicionar Afiliação de quaisquer Organizações

○ Administrador da organização: autorizado a criar/listar membros e dados

de suas Organizações

○ Usuário da organização: autorizado apenas a listar Organizações e

membros

○ Usuário anônimo: autorizado apenas a listar Organizações



Submissão



Em caso de dúvidas entre em contato e tire todas as dúvidas necessárias

Full Stack Developer Coding Challenge
Instruções gerais
Goal: Create a REST API that registers and lists Organizations, Users and

Membership of users to organizations.



Requirements



The following information is necessary for the registration and lookups of

each entity.



Organization:

● Name



User:

● Name

● Email



Membership:

● User

● Organization

● Role



The following operations should be available through the API:

● Should be able to register an Organization

● Should be able to return a list with all Organizations

● Should be able to return a single Organization with a specified ID

● Should be able to register a User

● Should be able to return a list of all Users

● Should be able to return a single User with a specified ID

● Should be able to register a User as Member of an specified

Organization

● Should be able to return a list of Members of a specified

Organization



For this task, we would like you to use the following tools:

● Serverless Framework preferably with Typescript

● DynamoDB database



Bonus

You can show your skill with these extra accomplishments if you want:

● Use a single table design on DynamoDB

● Use pagination for list results

● Write tests for the API

● Detailed input validation errors

● Auto-generated API documentation

● Authentication using OAuth2

● Create separate user roles with proper authorization enforcement on

the API:

○ System admin: authorized to create/list Organizations, Users

and add Membership of any Organizations

○ Organization admin: authorized to create/list members and data

of his/her Organizations

○ Organization user: authorized only to list Organizations and

members

○ Anonymous user: authorized only to list Organizations



Submission



If you have any doubts please get in touch and ask any necessary questions

before submitting your results.

When you are done, upload your code into a public Github repository with

short instructions on how to setup the system. Then send us the link to

that repository.
